# Grove Graphic Guides Reference

Complete templates and facilitation guides for strategic visual planning.

## Meeting Startup Templates

### River Rafting Format
Visual metaphor: Team navigating rapids together

**OARRs Framework:**
- **O**utcomes: What will we accomplish?
- **A**genda: What steps will we take?
- **R**oles: Who does what?
- **R**ules: How will we work together?

**Facilitation Steps:**
1. Draw river with rapids, calm sections, destination
2. Place team members in boat positions
3. Mark obstacles (rocks) and opportunities (calm pools)
4. Identify the destination (meeting outcome)

### Treasure Map Format
Visual metaphor: Journey to valuable destination

**Elements:**
- Starting point (current state)
- Treasure location (desired outcome)
- Path with landmarks (agenda items)
- Compass (guiding principles)
- Legend (roles and responsibilities)

## Context Map

**Purpose:** Early identification of critical trends and issues for proactive response.

**Template Structure:**
```
┌─────────────────────────────────────────────────────────────┐
│                    EXTERNAL ENVIRONMENT                      │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │   TRENDS    │  │   ISSUES    │  │ STAKEHOLDERS│         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
│                                                              │
│         ┌─────────────────────────────────┐                 │
│         │                                 │                 │
│         │      ORGANIZATION/PROJECT       │                 │
│         │                                 │                 │
│         └─────────────────────────────────┘                 │
│                                                              │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │OPPORTUNITIES│  │  CHALLENGES │  │  RESOURCES  │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
└─────────────────────────────────────────────────────────────┘
```

**Facilitation Steps:**
1. Introduce the process and explain the template
2. Brainstorm trends affecting the organization
3. Identify key issues and challenges
4. Map stakeholders and their interests
5. Discuss opportunities and resources
6. Synthesize findings into key themes

## SPOT Matrix

**Purpose:** Assess organization's current position within larger environment.

**Template Structure:**
```
┌─────────────────────────────────────────────────────────────┐
│                        SPOT MATRIX                           │
├─────────────────────────────┬───────────────────────────────┤
│        STRENGTHS            │         PROBLEMS              │
│                             │                               │
│  • Internal capabilities    │  • Internal challenges        │
│  • Core competencies        │  • Weaknesses                 │
│  • Resources available      │  • Gaps in capability         │
│                             │                               │
├─────────────────────────────┼───────────────────────────────┤
│       OPPORTUNITIES         │          THREATS              │
│                             │                               │
│  • External possibilities   │  • External risks             │
│  • Market openings          │  • Competitive pressures      │
│  • Emerging trends          │  • Regulatory changes         │
│                             │                               │
└─────────────────────────────┴───────────────────────────────┘
```

**Analysis Questions:**
- Which strengths can create opportunities?
- Which problems can create opportunities?
- Which threats cause problems?
- Which strengths mitigate threats?

## Cover Story Vision

**Purpose:** Draw on intuition and emotion to reveal what is important through future-perspective storytelling.

**Magazine Cover Format:**
```
┌─────────────────────────────────────────────────────────────┐
│  [MAGAZINE NAME]                              [FUTURE DATE] │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│                    [COVER IMAGE]                            │
│                                                              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              MAIN HEADLINE                           │   │
│  │         [Your Organization's Success Story]          │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                              │
│  • Subheadline 1: Key achievement                           │
│  • Subheadline 2: Impact on stakeholders                    │
│  • Subheadline 3: Innovation highlight                      │
│  • Pull quote: "..."                                        │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

**Facilitation Steps:**
1. Set future date (1-5 years out)
2. Imagine organization has achieved its vision
3. Write headlines as if reporting on success
4. Create pull quotes from key stakeholders
5. Describe the cover image
6. Share and discuss themes

## Five Bold Steps

**Purpose:** Define specific strategies that move organization toward vision.

**Template Structure:**
```
                        VISION
                          ▲
                          │
    ┌─────────────────────┼─────────────────────┐
    │                     │                     │
┌───┴───┐  ┌───┴───┐  ┌───┴───┐  ┌───┴───┐  ┌───┴───┐
│ STEP  │  │ STEP  │  │ STEP  │  │ STEP  │  │ STEP  │
│   1   │  │   2   │  │   3   │  │   4   │  │   5   │
└───────┘  └───────┘  └───────┘  └───────┘  └───────┘
    │          │          │          │          │
    ▼          ▼          ▼          ▼          ▼
[Action]   [Action]   [Action]   [Action]   [Action]
```

**For Each Step:**
- Bold statement of strategic initiative
- Key actions required
- Resources needed
- Timeline
- Success metrics
- Owner/champion

## Mandala Vision

**Purpose:** Group vision themes into unified whole, showing each facet as part of larger picture.

**Template Structure:**
```
                    ┌─────────┐
                    │ Theme 1 │
                    └────┬────┘
           ┌─────────┐   │   ┌─────────┐
           │ Theme 6 │───┼───│ Theme 2 │
           └────┬────┘   │   └────┬────┘
                │   ┌────┴────┐   │
                │   │         │   │
                ├───│  CORE   │───┤
                │   │ VISION  │   │
                │   │         │   │
                │   └────┬────┘   │
           ┌────┴────┐   │   ┌────┴────┐
           │ Theme 5 │───┼───│ Theme 3 │
           └─────────┘   │   └─────────┘
                    ┌────┴────┐
                    │ Theme 4 │
                    └─────────┘
```

**Facilitation Steps:**
1. Introduce the process
2. Vision storytelling (individual or small group)
3. Identify compelling themes
4. Cluster draft vision themes
5. Multi-vote on priorities
6. Refine and synthesize core vision

## Investment Portfolio

**Purpose:** Life-cycle framework for allocating critical resources.

**Template Structure:**
```
┌─────────────────────────────────────────────────────────────┐
│                   INVESTMENT PORTFOLIO                       │
├─────────────────────────────┬───────────────────────────────┤
│           SOW               │            GROW               │
│   (New Initiatives)         │    (Scaling Success)          │
│                             │                               │
│  • Emerging opportunities   │  • Proven initiatives         │
│  • R&D investments          │  • Expanding programs         │
│  • Pilot programs           │  • Market expansion           │
│                             │                               │
├─────────────────────────────┼───────────────────────────────┤
│          PLOW               │           HARVEST             │
│   (Maintain/Optimize)       │    (Wind Down/Extract)        │
│                             │                               │
│  • Core operations          │  • Mature products            │
│  • Efficiency improvements  │  • Declining markets          │
│  • Maintenance mode         │  • Resource extraction        │
│                             │                               │
└─────────────────────────────┴───────────────────────────────┘
```

**Allocation Questions:**
- What percentage of resources in each quadrant?
- Which initiatives should move between quadrants?
- What's the timeline for transitions?
- Who owns each initiative?
